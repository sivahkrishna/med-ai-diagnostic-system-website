# 🧠 MEDI AI – Smart Medical Diagnostic Web Application

## 📌 Overview

MEDI AI is an AI-powered web-based medical diagnostic system designed to assist healthcare professionals in detecting critical diseases from medical images. The system analyzes MRI, CT scan, and X-ray images using deep learning techniques to provide fast, accurate, and consistent diagnostic results.

The platform helps reduce manual workload, minimize human error, and improve accessibility to quality healthcare, especially in rural and resource-limited areas.

---

## 🚀 Key Features

* 🧠 Alzheimer’s disease detection from MRI scans
* 🦴 Bone fracture detection from X-ray images
* 🫁 Lung cancer detection from CT scan images
* 📤 Secure image upload and real-time prediction
* 📊 AI-based results with high accuracy
* 🏥 Hospital and admin management system
* 🔔 Notification and feedback system
* 📁 Patient result history tracking

---

## 🛠️ Technology Stack

* **Backend:** Python (Django Framework)
* **Frontend:** HTML, CSS, Bootstrap
* **Database:** MySQL
* **AI/ML:** Deep Learning (CNN – Convolutional Neural Networks)
* **Tools:** PyCharm, Android Studio

---

## ⚙️ How the System Works

1. Hospital uploads a medical scan image
2. The system preprocesses the image
3. AI model analyzes the scan using CNN
4. Disease prediction is generated
5. Result is stored and displayed to the user

---

## 🎯 Objective

The main objective of MEDI AI is to provide an intelligent diagnostic support system that assists doctors in early disease detection, reduces diagnosis time, and improves decision-making in healthcare.
